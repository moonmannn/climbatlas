# ClimbAtlas Approved Terminology

Use this glossary for public English and Chinese copy. Internal data labels may
remain in code and audit reports, but they must not appear as product language.

| Meaning | English | 中文 | Public rule |
| --- | --- | --- | --- |
| A climb in the catalog | Route | 路线 | Use consistently in navigation and cards. |
| A group of routes | Area / Sector | 区域 / 岩区 | Never present an area index as one route. |
| Published editorial selection | ClimbAtlas Pick | ClimbAtlas 精选 | Show only when editorial status is `published`. |
| Facts retained from a traceable source | Route facts | 路线信息 | Put source details in the Sources section. |
| Original grading text | Original grade | 原始难度 | Preserve the source wording. |
| Comparable local grading system | Primary grade system | 主要难度体系 | Do not imply an exact conversion. |
| Quiz-based alignment | DNA preference match | DNA 偏好匹配 | It is not an ability, safety, or readiness score. |
| Route attributes inferred from catalog facts | Inferred route profile | 推断的线路画像 | Explain the basis and limitation. |
| Link to one route | Route page | 线路页面 | The URL must resolve to that route. |
| Link to a guide or feature | Guidebook or feature | 路书或专题资料 | Do not copy the linked content. |
| Link to a broader climbing area | Area resource | 区域资料 | Make clear that it is not an exact route page. |
| Missing value | Not provided by source | 来源未提供 | Never replace with a guess or `TBD`. |

## Internal-only terms

Do not publish these labels:

- `pick candidate`
- `needs upgrade`
- `imported index`
- `metadata-only`
- `no outbound link`
- `editorial draft`
- `verification status`
- `completeness`
- raw trust levels or provider enums

## Product naming

- Use `Climbing DNA` for the preference quiz and profile.
- Use `Route Explorer / 线路探索器` for destination route browsing.
- Do not describe the retired Route Finder as a current product feature.
- Use `preference match / 偏好匹配` whenever a match percentage is visible.

## Source and editorial boundary

Route facts may be public when they pass the public route selector. Editorial
summaries, fit guidance, and route stories appear only after editorial status is
`published`. Missing editorial content must fall back to facts and sources, not
to generated filler.
